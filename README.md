Project Title: Student Performance Analysis and Prediction

Dataset: Student Performance Dataset (UCI Machine Learning Repository)

Source: https://archive.ics.uci.edu/ml/datasets/Student+Performance

Problem Statement:
Analyze how academic, social, and personal factors such as study time, family support, internet access, and number of absences influence student academic performance. Build a regression model to predict the final grade (G3).

Tasks Performed:
•	Exploratory Data Analysis to identify correlations between features and grades
•	Data cleaning and categorical feature encoding
•	Model development using Linear Regression and Random Forest Regression
•	Model evaluation using RMSE and R-squared metrics

Skills and Tools Used:
Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
